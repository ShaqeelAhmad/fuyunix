# fuyunix
A small platformer game using SDL2

# Dependencies
```
SDL2_image
gcc
```

# Portablility
I use Linux and can't guarantee that the program will work on other operating
systems

# Files
`src/` directory contains the source code
`man/` contains the man page
`data/` contains the assets

# Installation
```sh
make
make install
```
